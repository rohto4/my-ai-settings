# Registry

- `skills-registry.csv`: このrepoで独自作成・更新したskillの作者、日付、状態
- `skill-lineage.csv`: active 193件のgroup、移管元、frontmatter上のorigin、license・上流参照
- `upstreams.csv`: Git管理外でcloneした比較対象repositoryの固定commit、license、採用状態
- `skills-source-frontmatter.csv`: 移管前のfrontmatter原文
- `skills-classification.csv` / `skills-retention-review.csv`: 既存母集団の分類・保持評価

groupは保管経路、declared originは個別skillが名乗る出典を表す。両者を混同しない。
