# Codex Surface Map

| Need | Preferred surface | Do not use it for |
| --- | --- | --- |
| One task constraint | prompt or task context | durable project policy |
| Durable repository convention | `AGENTS.md` | long domain tutorials |
| Trusted project runtime setting | `.codex/config.toml` | secrets or user-global preference copies |
| Reusable focused workflow | skill | live external authorization |
| Shared installable bundle | plugin | one unfinished repository experiment |
| Live external data or actions | MCP server or app connector | static guidance already in docs |
| Deterministic repeated local operation | script or CLI | judgment-heavy product decisions |
| Mechanical lifecycle enforcement | hook | broad autonomous reasoning |
| Scheduled or recurring work | automation or persistent goal | immediate one-off action |
| Session-specific recovery | task ledger and handoff | replacing canonical specifications |

Check current product documentation before relying on exact paths, keys, or availability.
