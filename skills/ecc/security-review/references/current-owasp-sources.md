# Current OWASP Sources

Use these official OWASP sources when a review conclusion depends on a current
security requirement, a requirement identifier, or implementation guidance.
Retrieve the source at review time; do not treat a version, requirement ID, or
framework behavior recorded in this skill as current.

## Baseline and review method

- [OWASP ASVS project](https://owasp.org/www-project-application-security-verification-standard/)
- [OWASP Secure Code Review Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Secure_Code_Review_Cheat_Sheet.html)

Record the source URL and access date in a review only when the conclusion
depends on that external guidance. Cite ASVS controls with the version shown by
the current official release.
